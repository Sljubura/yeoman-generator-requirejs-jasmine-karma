# RequireJS Yeoman Generator

A generator for [Yeoman](http://yeoman.io).


## Getting Started

- Make sure you have [yo](https://github.com/yeoman/yo) installed:
    `npm install -g yo`
- Install the generator: `npm install -g generator-requirejs` *(locally works too)*
- Run: `yo requirejs`
